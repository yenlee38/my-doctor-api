## How to use

- Install for run:
	
npm install


- Create file .env :
  
HOST = localhost
	
USER = root
	
PASSWORD = password
	
DB = DatabaseName

JWT_SECRET = ?

- Also install nodemon globally, if you don't have it yet.

```sh
npm install -g nodemon
```

- Run project:
```sh
 nodemon server.js
```
